Banknote Authentication using Machine learning Algorithms

Banknote authentication is crucial for detecting counterfeit currency. This project compares the performance of several machine learning algorithms in authenticating banknotes: Decision Tree (DT), Artificial Neural Network (ANN), Backpropagation Neural Network (BPNN), K-Nearest Neighbors (KNN), and K-Means Clustering. Decision Tree (DT) achieved the highest accuracy among all models. The Decision Tree (DT) algorithm provided the most accurate and reliable results for banknote authentication, outperforming the other machine learning algorithms tested.
